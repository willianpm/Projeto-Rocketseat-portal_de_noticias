# Portal de Notícias

Projeto de interface desenvolvido como parte do currículo da Rocketseat para prática de fundamentos de desenvolvimento web.

## Descrição

O objetivo deste projeto é a criação de um layout de portal de notícias funcional, priorizando a organização semântica do conteúdo.

## Tecnologias

- HTML5 (Estruturação semântica)
- CSS3 (Grid Layout, Flexbox e Design Responsivo)

## Estrutura do Layout

- Cabeçalho com sistema de busca e navegação.
- Grade de notícias dividida entre destaques principais e colunas secundárias.
- Sidebar lateral para conteúdos relacionados.
- Rodapé informativo.
